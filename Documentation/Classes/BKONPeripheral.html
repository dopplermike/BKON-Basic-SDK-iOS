BKONPeripheral Class Reference

Inherits from NSObject
Conforms to CBPeripheralDelegate
Declared in BKONPeripheral.h

Overview

The BKONPeripheral class is used to interface with specific beacon after a communication channel has been established through the BKONCentralManager. It provides methods to modify a BKON beacons parameters.
